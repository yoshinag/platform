画像の輪郭抽出ツール

使用方法
  1. 画像をアップロードする（ファイル選択またはドラッグ＆ドロップ）
  2. 輪郭検出の感度を調整する
  3. 輪郭の太さを調整する（必要に応じて）
  4. プレビューで確認する
  5. 希望の形式でダウンロードする
    • 透過背景が必要な場合: PNG（透過）
    • 背景色が必要な場合: PNG（背景色指定）またはJPEG（背景色指定）
  6. サーバーへの画像アップロードは行われません。すべての処理はブラウザ上で実行されます。
