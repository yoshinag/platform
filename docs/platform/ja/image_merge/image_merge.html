画像の重ね合わせ＆ダウンロードツール

使用方法
  1. ベース画像をアップロードする（ファイル選択またはドラッグ＆ドロップ）
  2. オーバーレイ画像をアップロードする（ファイル選択またはドラッグ＆ドロップ）
  3. 必要に応じてオーバーレイ画像の位置とサイズを調整する
  4. 必要に応じて透明度を調整する
  5. プレビューで確認する
  6. 希望の形式でダウンロードする
    • 透過背景が必要な場合: PNG（透過）
    • 背景色が必要な場合: PNG（背景色指定）またはJPEG（背景色指定）
  7. サーバーへの画像アップロードは行われません。すべての処理はブラウザ上で実行されます。
