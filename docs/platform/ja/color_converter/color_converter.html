画像の色変換ツール

使用方法
  1. 画像をアップロードする（ファイル選択またはドラッグ＆ドロップ）
  2. 変換したい元の色を選択する
  3. 変換後の色を設定する
  4. 色の許容範囲を調整する（必要に応じて）
  5. プレビューで確認する
  6. 希望の形式でダウンロードする
    • 透過背景が必要な場合: PNG（透過）
    • 背景色が必要な場合: PNG（背景色指定）またはJPEG（背景色指定）
  7. サーバーへの画像アップロードは行われません。すべての処理はブラウザ上で実行されます。
