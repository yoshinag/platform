# platform

## 概要
dot_viewerは、DOT言語とMermaid記法に対応したグラフ描画・確認ツールです。このChrome拡張機能を使用すると、フローチャート、シーケンス図、ガントチャートなどの様々な種類のダイアグラムを簡単に作成、表示、編集することができます。

## 特徴
- DOT言語とMermaid記法の両方に対応
- テンプレートを使用して素早くダイアグラムを作成
- ダイアグラムをPNGまたはSVG形式でダウンロード
- コードのコピー、ペースト、クリア機能
- 全画面表示モード
- ユーザーフレンドリーなインターフェース

## 使い方
1. 拡張機能をインストールした後、Chromeツールバーのアイコンをクリックして起動します。
2. 記法（DOTまたはMermaid）を選択します。
3. テンプレートを選択するか、直接コードを入力します。
4. 入力したコードに基づいてグラフが自動的に描画されます。
5. 必要に応じて、ダウンロードボタンを使用してグラフをPNGまたはSVG形式で保存できます。

## 技術的詳細
- 外部ライブラリ：
    - Viz.js: DOT言語のレンダリングに使用
    - Mermaid.js: Mermaid記法のレンダリングに使用

## プライバシー
このツールはすべての処理をローカルで行い、入力されたコードは外部に送信されません。
